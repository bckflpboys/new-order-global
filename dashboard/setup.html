WhatsApp Web Not connected

The agent watches a WhatsApp group you create called "My Agent". Only messages in that group will trigger the agent or be visible to it. This keeps your agent isolated from your other chats.

Setup steps:
  1. Open WhatsApp on your phone → New Group → name it exactly My Agent → add only yourself (you can add a second number temporarily).
  2. Open web.whatsapp.com in this Chrome and scan the QR.
  3. Open the My Agent group once so it appears in your chat list.
  4. Click Verify below — the extension will confirm it can see the group.
